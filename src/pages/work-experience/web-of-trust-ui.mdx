**# Web of Trust UI Development Tools**

As a Frontend Developer & Open Source Maintainer at LinkedTrust, I developed two key tools for the Web of Trust ecosystem:
the `create-wot-app` CLI tool and the `@web-of-trust/ui` React component library.

<ExampleImage

**### create-wot-app CLI Tool**

The `create-wot-app` is a command-line interface tool designed to streamline the setup process for Web of Trust applications.
Key features include:

- Rapid project scaffolding
- Pre-configured Web of Trust components
- Built-in templates and best practices
- Automated dependency management

You can find the package on NPM [here](https://www.npmjs.com/package/create-wot-app).

**### @web-of-trust/ui React Library**

The `@web-of-trust/ui` is a comprehensive React component library that provides:

- Credential management components
- Authentication UI elements
- Advanced theming support
- Flexible data schema integration
- Reusable UI components

Check out the library on:

- [NPM](https://www.npmjs.com/package/@web-of-trust/ui)
- [GitHub](https://github.com/web-of-trust/ui)

**### Technologies Used**

- **React.js**: Building reusable UI components
- **TypeScript**: Ensuring type safety and better developer experience
- **Node.js**: CLI tool development
- **NPM**: Package distribution and management
- **Material-UI**: Theming and component base
